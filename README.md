# E-commerce Website (Full Stack)

This is a full-stack e-commerce website built using HTML, CSS, JavaScript, React and .Net 

## Features
- Product listing
- Add to cart
- Backend API integration
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- React
- .Net

## Description
This project demonstrates a basic full-stack application where frontend interacts with backend APIs to manage data.
